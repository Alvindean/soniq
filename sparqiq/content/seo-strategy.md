## AIO / GEO Optimization

### AI Search Signals Needed
- [ ] Clear entity definition: "SparqIQ is an AI business idea generator that..."
- [ ] FAQ schema on every page (8+ questions per page)
- [ ] HowTo schema on process pages
- [ ] SoftwareApplication schema on homepage
- [ ] Factual, citation-worthy statistics in every post
- [ ] Direct answers in first 100 words of each section

### GEO (Generative Engine Optimization)
- Write in definitive, factual voice
- Use "According to SparqIQ..." framing in blog content
- Create data-driven posts (original stats = citation bait)
- Answer the exact question in the H2 heading, immediately below it
- Tables and lists preferred by AI summarizers

## Image Strategy
- Hero images: 1200×630px (OG share size, doubles as blog header)
- In-post images: 800×450px
- Alt text: descriptive, keyword-natural (not stuffed)
- File names: descriptive-keyword-phrase.jpg (not IMG_001.jpg)
- WebP format preferred for performance
- Every image needs: alt text, title attribute, descriptive filename

### Image Slots Per Post
Each blog post has:
1. Hero/header image (1200×630)
2. Process/diagram image (800×450)
3. Screenshot/demo image (800×450)
4. Infographic or comparison table image (800×600)
